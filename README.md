# Кабинет кружка

Это электронный журнал для занятий, где есть:

- план работы;
- дневник занятий;
- учёт посещаемости;
- замечания проверяющих;
- краткий годовой отчёт;
- рабочий кабинет методиста;
- личный просмотр для родителя;
- личный просмотр для ребенка;
- тема занятия;
- посещаемость;
- домашнее задание.
- быстрый переход и просмотр с телефона;
- отправка ссылки родителю через WhatsApp.

## Как открыть

Откройте `index.html` в браузере.

## Как сделать интернет-ссылку

Если нужно, чтобы родители открывали журнал с телефона, его нужно опубликовать в интернете.

Готовая инструкция:

- [DEPLOY.md](C:/Users/UserNC/Desktop/Проекты/DEPLOY.md)

## Как пользоваться

1. Введите название группы и имя руководителя.
2. Добавьте детей и при желании контакт родителя.
3. Нажмите `Новое занятие`.
4. Укажите дату, время, тему, описание и домашнее задание.
5. Галочками отметьте, кто присутствовал.
6. При необходимости заполните `План работы`, `Замечания проверяющих` и `Краткий годовой отчёт`.
7. В блоке `Личные ссылки` выберите ребенка и нажмите нужную кнопку.
8. Можно сразу нажать `Отправить родителю в WhatsApp`.
9. Родитель откроет ссылку на телефоне и увидит посещаемость, тему и домашнее задание.

## Важно

Сейчас ссылка содержит снимок данных на момент копирования. Если вы изменили посещаемость или домашнее задание, лучше скопировать и отправить обновленную ссылку заново.

Для постоянной онлайн-ссылки без повторной отправки нужно вынести этот журнал в интернет и подключить общую базу данных.

## Следующий шаг

Если нужен настоящий онлайн-вариант с одной постоянной ссылкой и автоматическим обновлением для родителей, этот вариант можно доработать под сайт с общей базой данных и автоматической отправкой в мессенджер.
