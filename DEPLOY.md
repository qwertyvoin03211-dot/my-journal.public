# Как сделать рабочую ссылку для родителей

Сейчас журнал лежит у вас на компьютере, поэтому ссылка из файла не открывается у других людей.

Чтобы родители открывали журнал на телефоне, его нужно разместить в интернете.

## Самый простой способ: Netlify

Подходит, если нужна быстрая рабочая ссылка без сложной настройки.

Что сделать:

1. Откройте сайт [Netlify](https://www.netlify.com/).
2. Зарегистрируйтесь.
3. На главной панели выберите добавление нового сайта.
4. Выберите ручную загрузку файлов.
5. Загрузите папку с этими файлами:
   - `index.html`
   - `app.js`
   - `styles.css`
6. После загрузки Netlify выдаст интернет-ссылку вида `https://....netlify.app`
7. Именно эту ссылку уже можно отправлять родителям.

## Второй способ: GitHub Pages

Подходит, если хотите хранить сайт в GitHub.

Что сделать:

1. Создайте аккаунт на [GitHub](https://github.com/).
2. Создайте новый репозиторий.
3. Загрузите туда файлы проекта.
4. В настройках репозитория включите `Pages`.
5. GitHub выдаст ссылку на сайт.

## Что важно понимать

Сейчас проект уже подготовлен как статический сайт, то есть его можно выкладывать в интернет без сервера.

Но:

- пока данные хранятся только внутри браузера;
- для постоянной общей базы нужно отдельное подключение к серверу или базе данных;
- пока родителям лучше отправлять ссылку на конкретный просмотр после обновления данных.

## Что выбрать

Если нужна самая простая публикация:

- выбирайте `Netlify`

Если хотите потом развивать проект как сайт:

- выбирайте `GitHub Pages`
